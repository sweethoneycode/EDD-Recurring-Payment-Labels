Adds options to edit the labels for Recurring Payment products.
== Description ==
Have a download that uses Recurring payments, but you don’t know how to change the label?

This is exactly what this plugin does.

Simply upload it, change your label fields and now you’re rocking!

Your visitors will know what they are paying for!

This add editing fields under Easy Digital Downloads Extension Tab. 

These fields will enable you to edit the labels for Recurring Payment products.


*requires Recurring Payment extension found here: https://easydigitaldownloads.com/extensions/recurring-payments/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload ‘edd-shortcode-menu.php’ to the ‘/wp-content/plugins/‘ directory
1. Activate the plugin through the ‘plugins' menu in WordPress
1. Go to the Easy Digital Downloads settings and find the extension tab. You will see editing fields to edit the labels of products using Recurring Payments.
